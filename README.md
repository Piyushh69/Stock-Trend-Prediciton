# Stock-Trend-Prediciton
------------------------
Deployment - Streamlit

To predict Trends LSTM (Long Short-Term Memory) model is used. LSTMs are widely used for sequence prediction problems and have proven to be extremely effective. The reason they work so well is that LSTM can store past important information and forget the information that is not.

It performed well and a good model for sequence prediction problems.
